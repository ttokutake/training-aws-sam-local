# How to install aws-sam-local via asdf & nodejs

1. `$ asdf global nodejs <VERSION>`
2. `$ npm install -g aws-sam-local`

# How to start locally

1. `$ sam local start-api`
2. `$ curl http://localhost:3000/products`
